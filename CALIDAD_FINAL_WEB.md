## Los 12 cambios de paradigma (origen: crítica externa, 10-jul-2026, triados)
1. Que parezca un ser vivo, no una web (respira, crece, cambia solo)
2. El mapa ES la web entera (no una sección más)
3. Zoom infinito: MADRE → rama → idea → documento → decisión → código, sin cambiar de interfaz
4. Todo conectado — grafo de relaciones, no páginas sueltas
5. Menos texto, más comprensión visual (regla: si tarda >20s en entenderse, mal diseñado)
6. Explorar, no hacer scroll
7. Todo reacciona (tocar/volver/completar cambia algo visible)
8. Capas infinitas — mismo sitio sirve a niño, ingeniero e investigador
9. Más visual que textual (grafos/líneas de tiempo/dibujos como primer recurso, texto el último)
10. El visitante cambia MADRE de verdad (ve la consecuencia de su idea, no solo un "gracias")
11. Contar una historia (nací→pensé→me equivoqué→aprendí...), no 8 secciones
12. Efecto "no puedo dejar de explorar" (recompensa sorpresa cada 20-30s)

## Gates objetivos (marcar solo con prueba, no con percepción)
| Gate | Prueba |
|---|---|
| Comprensión | 5 personas entienden qué es MADRE en <30s |
| Curiosidad | ≥4 de 5 exploran más de una rama sin que se les pida |
| Mapa autoexplicativo | las 8 ramas se entienden SIN abrirlas |
| Identidad propia | 5 personas la describen con palabras parecidas ("mente", "organismo") |
| Recuerdo | al día siguiente recuerdan la idea principal |
| Navegación | nadie pregunta "¿por dónde empiezo?" |
| Claridad | una persona sin conocimientos técnicos sigue el recorrido principal |
| Valor | la mayoría dice haber aprendido/descubierto algo |
| Diferenciación | la mayoría dice que no se parece a una web de documentación normal |

**Criterio de cierre real:** la web es excelente cuando gente que NO conoce MADRE la explora
voluntariamente varios minutos, entiende la idea principal sin ayuda, y sale con ganas de volver o
enseñársela a otro. Eso se mide con personas reales, no releyendo este documento.
